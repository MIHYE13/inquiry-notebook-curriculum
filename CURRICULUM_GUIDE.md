# 📚 진도표 기능 사용 가이드

## 🎯 진도표 기능이란?

2025년도 4학년 과학 진도표를 기반으로, **차시별 탐구 노트를 빠르게 시작**할 수 있는 기능입니다.

### 기존 방식 vs 진도표 방식

| 기존 방식 | 진도표 방식 |
|----------|------------|
| 수동으로 주제 입력 | ✨ 차시 선택으로 자동 입력 |
| 핵심어를 직접 생각 | ✨ 핵심어 자동 제공 |
| 학습 목표 확인 필요 | ✨ 학습 목표 자동 연결 |
| 시간: 5-10분 | ⚡ 시간: 30초 |

## 📖 진도표 구성

### 4학년 1학기 (29개 차시)

#### 1단원: 힘과 우리 생활 (6차시)
```
1차시: 밀거나 당기며 느끼는 힘
  핵심어: 힘, 밀기, 당기기
  
2차시: 오르락내리락, 어느 것이 더 무거울까?
  핵심어: 무게, 비교, 시소
  
3차시: 도전, 무게를 정확히 비교하라!
  핵심어: 무게, 측정, 비교
  
4차시: 용수철저울로 무게를 비교하라!
  핵심어: 용수철저울, 무게, 측정
  
5차시: 작은 힘으로 물체를 들어 올리는 도구의 비밀
  핵심어: 도구, 지레, 힘
  
6차시: 일상생활에서 이용하는 도구를 찾아라!
  핵심어: 도구, 생활, 활용
```

#### 2단원: 물의 상태 변화 (5차시)
```
1차시: 물의 상태가 변할 수 있다고?
  핵심어: 물, 상태, 변화
  학습 목표: 물의 세 가지 상태를 이해한다.
  
2차시: 물이 얼 때와 얼음이 녹을 때 어떤 일이 일어날까?
  핵심어: 얼음, 얼기, 녹기
  
3차시: 물이 증발할 때와 끓을 때 어떤 일이 일어날까?
  핵심어: 증발, 끓음, 수증기
  
4차시: 차가운 물체 표면에 어떤 일이 일어날까?
  핵심어: 응결, 이슬, 물방울
  
5차시: 물을 얻는 방법을 조사하자!
  핵심어: 물, 얻기, 방법
```

#### 3단원: 땅의 변화 (5차시)
```
3차시: 화산이 궁금해!
  핵심어: 화산, 마그마, 분출
  학습 목표: 화산의 구조와 분출 과정을 이해한다.
  
4-7차시: 화산 활동 모형, 화강암과 현무암, 화산의 영향, 지진
```

#### 4단원: 식물의 생활 (5차시)
```
1차시: 식물 탐험대, 식물을 분류하라!
  핵심어: 식물, 분류, 특징
  학습 목표: 식물을 특징에 따라 분류할 수 있다.
  
2-6차시: 주변 식물, 물 속 식물, 사막/산 식물, 생활용품
```

#### 5단원: 생물의 한살이 (3차시)
```
4차시: 씨가 싹 트려면 어떤 조건이 필요할까?
  핵심어: 씨앗, 싹트기, 조건
  학습 목표: 씨가 싹트는 조건을 실험으로 알아본다.
  
5-7차시: 식물 성장, 한살이
```

#### 6단원: 다양한 생물과 우리 생활 (5차시)
```
1-6차시: 버섯/곰팡이, 해캄/짚신벌레, 세균, 영향, 생명과학
```


### 차시 정보 표시

각 차시 카드에는:
- ✅ 차시 번호 (파란 배지)
- ✅ 차시 주제 (굵은 글씨)
- ✅ 대단원 (📚 아이콘)
- ✅ 핵심어 (보라색 태그)
- ✅ 학습 목표 (회색 박스)

## ⚙️ 기술 구현

### 데이터 구조

```typescript
// 진도표
type Curriculum = {
  id: 'curriculum-4-1-2025';
  name: '2025년도 4학년 1학기 과학';
  grade: '4학년';
  subject: '과학';
  semester: '1학기';
  lessons: CurriculumLesson[];
}

// 차시
type CurriculumLesson = {
  id: 'lesson-012';
  unit: '땅의 변화';
  topic: '화산이 궁금해!';
  period: 3;
  keywords: ['화산', '마그마', '분출'];
  learningGoals: '화산의 구조와 분출 과정을 이해한다.';
}

// 탐구 노트 (진도표 정보 추가)
type InquiryEntry = {
  // ... 기존 필드들 ...
  curriculumInfo?: {
    lessonId: 'lesson-012';
    unit: '땅의 변화';
    topic: '화산이 궁금해!';
    period: 3;
  }
}
```

### 자동 입력 로직

```typescript
function handleLessonSelect(curriculum, lesson) {
  const entry = {
    date: getTodayDate(),
    todayTopic: lesson.topic,  // 자동 입력
    questions: `${lesson.topic}에 대해 궁금한 점을 적어보세요.\n\n핵심어: ${lesson.keywords.join(', ')}`,  // 자동 입력
    priorKnowledge: `학습 목표: ${lesson.learningGoals}`,  // 자동 입력
    curriculumInfo: {
      lessonId: lesson.id,
      unit: lesson.unit,
      topic: lesson.topic,
      period: lesson.period
    }
  };
  
  setCurrentEntry(entry);
  showToast('탐구 노트가 준비되었습니다! ✨');
}
```

## ❓ FAQ

### Q1: 진도표를 직접 추가할 수 있나요?
A: 현재 버전(v2.0)에는 4학년 1, 2학기만 내장되어 있습니다. v2.1에서 업로드 기능이 추가될 예정입니다.

### Q2: 차시 순서를 바꿀 수 있나요?
A: 진도표는 교육과정 순서대로 고정되어 있습니다. 원하는 차시를 자유롭게 선택할 수 있습니다.

### Q3: 진도표 없이도 사용할 수 있나요?
A: 네! 기존처럼 직접 주제를 입력할 수 있습니다. 진도표는 선택사항입니다.

### Q4: 다른 학년도 추가되나요?
A: v3.0에서 5, 6학년 추가 예정입니다.
